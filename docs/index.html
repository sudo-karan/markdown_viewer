      Markdown Studio

      A private, offline-friendly Markdown editor that runs entirely in your browser — no server, no tracking.

      Shortcuts

      Ctrl/Cmd + SSave (Drive if connected, else local)
      Ctrl/Cmd + Shift + SSave a copy to Google Drive
      Ctrl/Cmd + Shift + NNew document
      Ctrl/Cmd + B / I / KBold / Italic / Link
      Ctrl/Cmd + 1 / 2 / 3Edit / Split / Preview view
      Tab in a listContinues the list / indents
      EscClose dialogs

      Features

      • GitHub-flavored Markdown: tables, task lists, footnotes, alerts (> [!NOTE])
      • Syntax highlighting, Mermaid diagrams, and KaTeX math ($…$, $$…$$)
      • Live split preview with synced scrolling and a clickable outline
      • Local-first: autosaves to your browser; open/save files from disk
      • Google Drive sync and an optional activity log to your own Sheet
      • Paste an image to embed it, and share a doc as a self-contained link
